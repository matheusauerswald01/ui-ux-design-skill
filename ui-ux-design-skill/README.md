## ✨ O que é?

Uma **skill** instrui o Claude a agir como um designer/engenheiro de interfaces de nível mundial. Esta é a versão mais abrangente — cobre desde psicologia cognitiva até shaders GLSL, de OKLCH a physics-based animations, de fluid simulation a design ético.
