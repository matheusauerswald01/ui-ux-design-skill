# Changelog

Todas as mudanças notáveis neste projeto serão documentadas aqui.

Formato baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/).

## [1.0.0] — 2025

### Adicionado
- SKILL.md principal com raciocínio de layout preventivo
- Sistema tipográfico completo com `clamp()` fluido
- Padrões de animação para GSAP, Framer Motion, Three.js e Remotion
- Componentes premium: magnetic button, text reveal, glass card, noise texture, cursor
- Referência de técnicas avançadas (parallax, page transitions, shaders, marquee)
- Sistema de paletas de cores por categoria (5 paletas base)
- Checklist de entrega e performance
- Documentação completa com guia, prompts prontos e FAQ
